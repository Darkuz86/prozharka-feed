# prozharka-feed

Краткое описание
- Репозиторий для страниц с RSS‑лентами публикаций.
- Здесь будут храниться и генерироваться страницы/файлы RSS для разных источников и публикаций.

Как использовать (минимально)
- Каждая лента — отдельная страница/файл в репозитории (например, в папке `feeds/`).
- Формат — стандартный RSS (XML).
- Добавьте или обновите файл ленты и создайте PR.

Цель проекта
- Собрать и предоставить простые публичные RSS‑ленты для публикаций на одном месте.

Контрибуция
- PR принимаются. Описание изменений в PR + пример ссылки на тестовую ленту.

Лицензия
- Укажите лицензию в отдельном файле (например, `LICENSE`) по желанию.
